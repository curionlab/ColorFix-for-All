# ColorFix for All

> **色をなおす、デザインを守る。**  
> 色相を固定した探索と数理最適化によって、PDF/画像のカラーアクセシビリティ改善を支援するツールです。

[**🚀 Live Demo を試す**](https://curionlab.github.io/ColorFix-for-All/)

ColorFix for All は、**色彩工学と数理最適化**を用いてドキュメントのカラーアクセシビリティを評価・修正するツールです。WCAG 2.2 や ISO 24505-2:2025 の考え方を参照し、ブランドカラーの色相を保ちながら、視覚的な識別性を高める代替カラーを提案します。

---

## 主要機能

### 1. ブラウザ内での解析
- **クライアントサイド処理**: 解析処理はブラウザ内で完結する構成を採用しています。
- **OCR対応**: `Tesseract.js` を用いて、テキスト情報を持たないスキャン済みPDFや画像（PNG/JPG）から文字を抽出し、アクセシビリティ解析に利用できます。

### 2. OKLCH による制約付き最適化
- **色相を保った探索**: OKLCH 色空間を用い、元の色の色相を固定したまま候補色を探索します。
- **ブランドイメージへの配慮**: 通常視および色覚特性シミュレーション下でコントラスト条件を満たしつつ、元の色との差が小さい候補を算出します。

### 3. 複数の評価基準に対応
- **WCAG 2.2**: 相対輝度に基づくコントラスト比判定（AA/AAA）。
- **ISO 24505-2 を参考にした評価**: P型・D型・T型の色覚シミュレーションや、ロービジョンを意識した明度差指標を用いて識別性を評価します。

## 計算基準と理論的背景

### 1. OKLCH の採用
色の調整において、見た目の変化を扱いやすい色空間として **OKLCH** を採用しています。これにより、アクセシビリティ対応のための調整でも、元の色の印象をできるだけ保った探索がしやすくなります。

### 2. 知覚色差の評価
提案色と元の色の差を評価するために、知覚色差式 **CIEDE2000** を採用しています。色の近さを、人間の見え方に配慮した形で数値化します。

### 3. 制約付き最適化
- **目的関数**: 元の色との知覚色差 \(\Delta E_{00}\) を最小化。
- **制約条件**: 通常視および CVD 3種（P/D/T型）で、目標コントラスト比を満たすこと。

### 4. 用途別の判定指標
- **Semantic（凡例・図表カテゴリなど）**: 色そのものによる識別を重視し、最小知覚色差と明度差を指標にします。
- **Text（文字・アイコン）**: 上記に加えて、コントラスト比 4.5:1 以上を可読性の目安として適用します。

### 5. CVD シミュレーション
P型（Protanopia）・D型（Deuteranopia）・T型（Tritanopia）の推定には、混同線に基づく行列変換を用いたシミュレーションを採用しています。シミュレーション後の色空間で、色差やコントラストを再評価します。
